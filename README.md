# Перевод баллов ОГЭ (русский язык)

Небольшой учебный проект с приветственным лендингом и калькулятором перевода первичных баллов ОГЭ по русскому языку в итоговую школьную оценку.

## Что умеет сайт

- показывает мини-лендинг с коротким описанием проекта;
- рассчитывает итоговую оценку по общему баллу и критериям ГК1-ГК4;
- проверяет корректность введенных значений;
- выводит пояснение по результату расчета.

## Структура проекта

- `index.html` — основная страница;
- `css/style.css` — стили лендинга и калькулятора;
- `js/app.js` — логика валидации и расчета.

## Как запустить

1. Откройте папку проекта.
2. Запустите файл `index.html` в браузере.

Если хотите локальный сервер, можно использовать расширение Live Server в VS Code/Cursor.
